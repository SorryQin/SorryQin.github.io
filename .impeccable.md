# Impeccable Design Context

## Design Context

### Users
- **Primary**: 招聘官、教授、合作研究者（评估学术/技术能力）
- **Secondary**: 技术社区同行、技术招聘者（评估技术实力）
- **Tertiary**: 校友、朋友、家人（了解近况）
- **Context**: 多为线上浏览，在桌面或移动设备上快速扫描，寻求第一印像

### Brand Personality
- **Personality**: 专业、技术实力、年轻活力
- **Voice**: 简洁直接，数据驱动，强调成果
- **3 Words**: 技术、精准、前沿

### Aesthetic Direction
- **Theme**: 深色科技风（Dark Mode）
- **Mood**: 专业但不冰冷，展现AI/Computer Vision领域的前沿感
- **Visual Tone**: 类似顶级AI实验室/科技公司官网，高端技术产品

### Design Principles
1. **技术感优先**: 深色背景，清晰的数据展示，代码/终端风格元素
2. **成果导向**: 突出论文、项目、数据指标，而非长段描述
3. **现代简约**: 避免过度装饰，保持专业感
4. **性能意识**: 快速加载，流畅动画，无冗余元素
5. **响应式完整**: 桌面和移动端体验同样重要

### Color Palette
- Primary: #4f46e5 (紫色 - 保持原方案)
- Secondary: #ec4899 (粉色 - 用于强调)
- Accent: #10b981 (翠绿 - 用于成功/活跃状态)
- Background: #0f172a (深蓝黑)
- Surface: #1e293b (深灰)
- Text: #f8fafc (亮白)
- Muted: #94a3b8 (灰色文字)

### Typography
- Display: 技术感强的高质量无衬线字体
- Body: 清晰易读的现代字体

### Layout
- 卡片式信息展示
- 时间线展示经历
- 项目/论文网格布局
- Hero区域展示核心身份
