## Design Context

### Users
Visitors viewing Sorry Qin's personal portfolio - potential employers, collaborators, academics, and fellow researchers in AI/Computer Vision.

### Brand Personality
Tech-forward, creative, detail-oriented. Cyberpunk/synthwave aesthetic reflecting interest in technology and futuristic design. Personal but professional.

### Aesthetic Direction
**Cyberpunk/Synthwave** - Dark backgrounds (#0a0a12), neon accents (cyan #00ffff, purple #8a2be2, magenta #ff00ff), grid patterns, glowing borders, 3D neural network visualization.

### Design Principles
1. **Neon on dark**: All accent colors should glow, never flat colors on dark backgrounds
2. **Consistent spacing**: Use consistent padding/margins with visual rhythm
3. **Hover states**: All interactive elements should have clear hover feedback with glow effects
4. **Accessibility**: Maintain WCAG AA contrast, visible focus indicators, 44px touch targets
5. **No side-stripe borders**: Blockquotes and callouts should NOT use left border stripes (BAN 1 violation)
