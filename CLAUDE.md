# Sorry Qin Personal Website

## Project Overview
Personal portfolio website for Sorry Qin (覃佐睿), showcasing academic background, research work, and personal interests.

## Tech Stack
- Static site (Jekyll compatible markdown + HTML)
- CSS with custom properties for theming
- Google Fonts for typography

## Design Context

### Users
- **Primary**: 招聘官、教授、合作研究者
- **Secondary**: 技术社区同行、技术招聘者
- **Tertiary**: 校友、朋友、家人

### Brand Personality
- 专业、技术实力、年轻活力
- 3 Words: 技术、精准、前沿

### Aesthetic Direction
- **Theme**: 深色科技风（Dark Mode）
- **Mood**: 专业但有活力，展现AI/Computer Vision领域前沿感
- **Reference**: 顶级AI实验室/科技公司官网风格

### Design Principles
1. 技术感优先：深色背景，清晰数据展示
2. 成果导向：突出论文、项目、数据指标
3. 现代简约：避免过度装饰
4. 性能意识：快速加载，流畅动画
5. 响应式完整：桌面和移动端同等重要

### Color Palette
- Primary: #4f46e5 (紫)
- Secondary: #ec4899 (粉)
- Accent: #10b981 (翠绿)
- Background: #0f172a (深蓝黑)
- Surface: #1e293b (深灰)
- Text: #f8fafc (亮白)
- Muted: #94a3b8 (灰色)

### Content Structure
- Home: Hero + About + Timeline + Contact
- Publications: 论文列表
- Awards: 奖学金/荣誉
- Hobbies: 篮球、健身、宠物
- Blogs: 博客文章列表
- Startups: 创业相关内容

## Commands
```bash
# Local development (if Jekyll)
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```
